# Health-Check Script 

Script en Bash para verificar rápidamente el estado (HTTP y tiempos de respuesta) de una lista de URLs y generar un reporte en CSV.

#  Uso 

# Prepara tu archivo de entrada
Crea un archivo de texto (ej. `urls.txt`) y coloca una URL por líneas

# Da permisos de ejecución
Abre tu terminal en la carpeta del script y ejecuta:

chmod +x health_check.sh

# Ejecuta el script
./health_check.sh urls.txt
